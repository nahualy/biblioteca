#  Biblioteca Digital

Sistema de búsqueda y gestión de libros desarrollado con JavaScript vanilla y Bootstrap.

## 📋 Descripción

Aplicación web que permite buscar libros de acuerdo a sus características (ISBN, nombre, autor y categoría). Los datos se almacenan en un archivo JSON que actúa como base de datos local.

## ✨ Características

-  Búsqueda por múltiples criterios (ISBN, nombre, autor, categoría)
- ✅ Validaciones de campos de entrada
- 📱 Diseño responsive con Bootstrap 5
- 🎨 Interfaz moderna y minimalista
- 📊 Visualización de resultados en tabla
-  Función para limpiar filtros de búsqueda


## ️ Tecnologías

- HTML5
- CSS3 (con Bootstrap 5.3.7)
- JavaScript (ES6+)
- JSON (almacenamiento de datos)

##  Base de Datos

El archivo `libros.json` contiene 20 libros con la siguiente información:

- **ISBN**: Identificador único del libro
- **Nombre**: Título del libro
- **Autor**: Nombre del autor
- **Edición**: Número de edición
- **Categoría**: Género o categoría del libro

## 🚀 Instalación

1. Clona o descarga el repositorio
2. Abre el archivo `index.html` en tu navegador
3. ¡Listo! No requiere instalación de dependencias

##  Uso

1. Ingresa los criterios de búsqueda en los campos correspondientes:
   - ISBN
   - Nombre del libro
   - Autor
   - Categoría (selector desplegable)

2. Haz clic en **"Buscar"** para filtrar los libros

3. Usa **"Limpiar"** para resetear los filtros y mostrar todos los libros

## ✅ Validaciones

El sistema incluye validaciones para:
- Campos vacíos
- Formato de ISBN
- Caracteres especiales en nombres
- Selección de categoría válida
